# キャプチャ FPS と画質の関係

`cam_ui.py` と `web_ui.py` の両方で検証した結果、Jetson カメラパイプラインでは **キャプチャ FPS** が見た目の画質を左右する支配的な要素であることが分かっています。

- **15 FPS キャプチャ**: 画面は非常にクリーンでノイズもほとんど目立ちません。JPEG 品質を 80〜85 に設定すれば、破綻のない安定したフレームが得られます。
- **30 FPS キャプチャ**: 特に暗部でセンサノイズが増え始めます。JPEG 品質を高めても圧縮ノイズが混じりやすく、全体的にざらついた印象になります。
- **60 FPS キャプチャ**: ノイズと色ムラが顕著になり、画質が大きく劣化します。高速読み出しによってセンサノイズが増幅され、エンコード処理も逼迫するため、JPEG の調整では補えないレベルで品質が落ちます。

**推奨:** `cam_ui` の画質検証やセグメンテーション結果の比較など、鮮明な映像が必要な場合は **15 FPS** のまま運用してください。動作遅延を優先して FPS を 30/60 に引き上げる場合は、必ずその画質低下を想定・記録したうえで使用します。
